# Portail Analyse Genre - GitHub Pages

## Objectif
Portail statique moderne pour explorer le contexte de l’étude, l’analyse genre et l’analyse des organisations ODF/ODFF.

## Architecture
- `index.html` : contexte de l’étude et architecture.
- `analyse-genre.html` : filtres et visualisations répondant.e.s.
- `organisations.html` : analyse des ODF/ODFF.
- `donnees.html` : catalogue et liens de téléchargement.
- `assets/js/utils.js` : fonctions robustes communes.
- `assets/js/app.js` : logique des pages.
- `data/portal-data.js` : données consolidées intégrées.
- `data/*.csv` : bases standardisées.

## Déploiement GitHub Pages
1. Créer un nouveau dépôt GitHub.
2. Décompresser ce package à la racine du dépôt.
3. Pousser les fichiers sur la branche `main`.
4. Aller dans `Settings > Pages`.
5. Choisir `Deploy from branch`, branche `main`, dossier `/root`.

## Données
Les microdonnées publiées sont dé-identifiées. Pour un portail public, éviter de publier les noms d’enquêteurs, répondant.e.s, contacts ou organisations nominatives.
